How the app learns

The grinder model now treats your machine program as a fixed-time shot. It learns how yield changes with grinder setting, then uses yield error to decide direction: low yield means go coarser, high yield means go finer.

Dose is adjusted separately through grind time. Recommendation priority is always micro first, then macro, then grind time.
